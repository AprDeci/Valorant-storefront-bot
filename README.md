# Valorant-storefront-bot
 自动推送valorant每日商城telegrambot

使用.env作为变量


采用webhook,polling请自行更改

请创建.env文件并且输入以下信息

~~~env
PORT=''
TOKEN=''
URL=''
User=''
PASSWORD=''
~~~
